# ADR-014 — Shadow Mode First

## Status

Accepted.

## Context

Agency Terminal should not become politically authoritative before reviewers and leadership trust the scoring model.

## Decision

The bot launches in shadow mode: real evidence and reviews, officer-only profiles, no automatic authority impact.

## Consequences

- Metrics can be calibrated safely.
- Officers can test workload.
- Members are not immediately judged by untested scores.
- Exit criteria are required before advisory or operational use.
