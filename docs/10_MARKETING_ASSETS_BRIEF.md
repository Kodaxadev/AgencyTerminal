# Marketing Assets Brief

## Visual direction

Agency Terminal should feel like a secure ops interface, not a SaaS dashboard.

Keywords:

- Black glass terminal
- Amber/green command text
- Radar-grid overlays
- Evidence ledger cards
- Mercenary contract case files
- Signal scanlines
- Redacted dossiers
- Code 200 / Code 403 status language
- Minimal color, high contrast

## Suggested sizes

- Hero: 1600×900
- Social card: 1200×630
- Discord announcement: 1600×900
- Square icon/poster: 1080×1080
- Section card graphics: 1200×800

## Palette

- Background: near-black / charcoal
- Primary terminal text: muted green
- Secondary: amber/gold
- Danger/rejected: restrained red
- Borders: translucent white/green
- Accent: signal cyan sparingly

## Typography

Use site-adjacent terminal styling:

- Mono for system labels and status codes
- Condensed/square sans for headers
- Small uppercase labels
- Wide letter spacing
